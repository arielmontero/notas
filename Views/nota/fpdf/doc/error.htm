Error

Error(string msg)

Descripción

Este método es automáticamente invocado en caso de un error fatal; este simplemte emite el mensaje y para la ejecución. Una clase heredada puede anularlo para personalizar el manejo de errores pero siempre debería parar el script, o el resultado probablemente sería no válido.

Parámetros

msg
El mensaje de error.
